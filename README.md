### What I learned

- Practiced **responsive design** using media queries
- Used **Flexbox** to properly stack and center elements vertically inside the card
- Correctly implemented **Google Fonts** (Sora) with preconnect links
- Applied consistent styling with **border-radius**, **box-shadow**, and color palette
- Improved semantic HTML structure
- Created consistent spacing and typography using margins and the chosen font family

### Useful resources

- [Google Fonts - Sora](https://fonts.google.com/specimen/Sora) — Beautiful modern sans-serif font used throughout the project
- [devChallenges - Minimal Blog Card](https://devchallenges.io/challenge/minimal-blog-card) — Official challenge page with design reference

### Built with

- Semantic HTML5 markup
- CSS (vanilla – no framework)
- Flexbox
- Media Queries (for responsiveness)
- Google Fonts

## Features

- Minimal blog card layout with image, category tag, title, description, and author name
- Fully responsive design (adjusts card width and image size on different screen sizes)
- Soft shadow and rounded corners for a modern, clean look
- Clean and modern typography using the Sora font
- Centered card layout on the page

This application/site was created as a submission to a [DevChallenges](https://devchallenges.io/challenges-dashboard) challenge.
